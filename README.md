# RED Lab

Public lab — published prototypes, guides, and experiments from RED.
Served at **[lab.slash.red](https://lab.slash.red)** via GitHub Pages.

## Scope — public only

This repo is **public**. Only public-intended content lives here. RED-internal
work, client work, and the marketing engine stay private (`red-os` repo,
`antimofm/lab-archive`). Nothing confidential. No secrets, no `.enc.json`,
no client material — `.gitignore` enforces the obvious cases; judgment covers
the rest.

Publishing here is a deliberate, gated step — not an automatic sync from any
internal system. See `red-os/marketing/buildout-plan.md`.
